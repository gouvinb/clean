# clean
Clean directory of temp files generate.

## Installation
- Go [https://golang.org](https://golang.org)
- `go install github.com/gouvinb/goclean`

## Usage
### Usage of clean :
Into terminal :

`clean [FLAG arg...] <directory>`

### List of flag
- `all`
  - clean at all level
- `pattern`
  - add pattern
